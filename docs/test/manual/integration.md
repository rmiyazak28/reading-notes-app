## 不合格時の記録欄

不合格（❌）となったケースは以下に記録する。

| # | 内容 | 対応状況 |
|---|---|---|
| 1-1 | サマリーバーの数値が変わらない | 修正済み。totalBooks・readingBookCount を useState で管理し、handleBookCreated で書籍登録時に両方インクリメント（読書状態「読書中」のときのみ readingBookCount を加算するように変更。HomeSummaryBar に state 値を渡すよう変更。 |
| 5-3 | 絞り込まれたままの全メモ検索画面に戻らない | 修正済み。 memo-search-card.tsx: useSearchParams() で現在の検索条件（q=xxx&favorite=1 など）を取得し、from パラメータにフルパス（例: /memos?q=キーワード）を URL エンコードして渡すよう変更。edit/page.tsx: from を decodeURIComponent でデコードし、/memos または /books/ で始まる相対パスであれば backTo としてそのまま使用（外部 URL 注入を防ぐバリデーション付き）。 |
