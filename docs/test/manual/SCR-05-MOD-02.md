## 不合格時の記録欄

不合格（❌）となったケースは以下に記録する。

| # | 内容 | 対応状況 |
|---|---|---|
| 1-1 | ジャンル・ステータス・登録日が一列に並んでいて見づらい | 修正済み。2段に変更 |
| 1-1 | ★のところが「★★重要２件」となっている。「お気に入り」ではなく、「重要」になっている | 修正済み。「★ お気に入り ●件」へ変更。重要→お気に入りに変更。 |
| 2-3 | 1行で省略時に...がない | 修正済み。2行でline-clamp-2に変更。 |
| 6-3 | 削除時にお気に入り件数が減らない | 修正済み。StarCountを減らすように変更。 |
| 8-2 | 1回項目を修正して、更新を押さずにモーダルを閉じて、再度表示すると修正内容が残ったまま | 修正済み。再表示も現在の書籍情報が初期値で設定されるように変更。 |
