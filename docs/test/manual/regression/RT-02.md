# memoLake RT-02　リグレッションテストシナリオ

本ドキュメントは、searchMemos の検索方式を RPC（`search_memos`）から `reading_memos.search_text` カラム + GIN インデックス（`.ilike()`）へ移行したことに伴い、再実施が必要なマニュアルテストシナリオをまとめたものです。

---

## 対象変更

| 項目 | 変更前 | 変更後 |
|---|---|---|
| 検索実装 | `supabase.rpc("search_memos", ...)` | `.from("reading_memos").select(...).ilike("search_text", ...)` |
| 検索対象 | RPC内でJOIN（content / book.title / book.author / tags.name） | `search_text` カラム（content + 書籍名 + 著者名 + タグ名をトリガーで非正規化） |
| インデックス | 複数テーブル横断（GINインデックス未使用） | `idx_memos_search_text_trgm`（GIN + pg_trgm） |
