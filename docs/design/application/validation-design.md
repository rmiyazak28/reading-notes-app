# 7. バリデーション設計

## 7.1 書籍登録

| 項目     | 条件                                        |
| ------ | ----------------------------------------- |
| タイトル   | 必須 / 255文字以内                              |
| 著者     | 任意 / 255文字以内                              |
| ジャンル   | 任意 / 100文字以内                              |
| 読書状態   | 任意 / unread・reading・completed のいずれか（デフォルト：unread） |

---

## 7.2 書籍編集

| 項目     | 条件                                        |
| ------ | ----------------------------------------- |
| タイトル   | 必須 / 255文字以内                              |
| 著者     | 任意 / 255文字以内                              |
| ジャンル   | 任意 / 100文字以内                              |
| 読書状態   | 任意 / unread・reading・completed のいずれか       |
| 読了日    | status が "completed" の場合のみ表示・必須 / それ以外は非表示 |

---

## 7.3 読書メモ登録

| 項目     | 条件                        |
| ------ | ------------------------- |
| ページ数   | 任意 / 1以上の整数               |
| メモ内容   | 必須 / 5000文字以内             |
| タグ     | 任意 / 各要素はUUID形式           |
| お気に入り  | 任意 / boolean（デフォルト：false） |

---

## 7.4 タグ登録

| 項目   | 条件            |
| ---- | ------------- |
| タグ名  | 必須 / 50文字以内   |

---

## 7.5 ユーザー登録

| 項目       | 条件              |
| -------- | --------------- |
| ユーザー名    | 必須              |
| メールアドレス  | 必須 / メール形式      |
| パスワード    | 必須 / 8文字以上72文字以内 / 英字・数字をそれぞれ1文字以上含む              |
| パスワード確認  | 必須 / パスワードと一致   |

---

## 7.6 プロフィール更新

| 項目      | 条件         |
| ------- | ---------- |
| ユーザー名   | 任意         |
| メールアドレス | 任意 / メール形式 |
| パスワード   | 任意 / 変更する場合は8文字以上72文字以内 / 英字・数字をそれぞれ1文字以上含む         |
| パスワード確認  | パスワードが変更される場合は必須 / 新しいパスワードと確認用パスワードの一致   |
